Rename your geckdriver depending of your O.S
For instance, if you use Windows-32, rename
geckodriver-32.exe to geckdriver.exe

If you are using MacOS or Windows64, you don't
have to change the names.

If you are using Linux, rename the file geckodriver
to geckdriver-macos (default) and choose
your linux geckodriver-linux32(or 64) to geckdriver
